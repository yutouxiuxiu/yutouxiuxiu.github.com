页面定时跳转

{ post:time }
常用技巧 页面定时跳转的js实现方式 要加一个如果没有自动跳转请点击这里（防止因为页面太大 没有加载完 它无法自动跳转 onload是在加载完之后执行的）
<head>

<meta http-equiv="Refresh" content="秒数;目标url" />

<script type="text/javascript">

var second = 5;

function init() {

//获得到指定元素

var secondSpan = document.getElementById("second");

//获取标签中内容

secondSpan.innerHTML = second;

second--;

//定时执行

window.setTimeout('init()',1000);//每多少毫秒执行一次这个方法或表达式

}

</script>

</head> <body onload="init();">

<!--js是事件驱动的 然后调用一个方法 它一启动就执行 所以用onload事件-->

<h3>页面将在<span id="second">5</span>后跳转</h3>

</body>

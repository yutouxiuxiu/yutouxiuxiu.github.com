一、数组的创建

key 可以是数字/字符串 

key为纯数字叫索引数组 其他较关联数组

value 可以是任何类型的

 

如果没有指定键 会以当前存在的整数键+1 作为新的键

如果当前没有整数键 从0开始

如果指定的键已经有值 则将被覆盖

1.$array = array('' => '', ...);

2.$array = array('', ...);

3.$array[''] = '';

4.$array[] = '';  //会自动向后开辟一个新的键

5.$array = array('' => '', '', ...);

$array[] = 'a';
$array[3] = 'b';
$array[] = 'c';
$array[] = 'd';

//（结果！！！）Array ( [0] => a [3] => b [4] => c [5] => d )

$array = array('a', 'b', 'c', 'd');
unset($array[3]);
$array[] = 'e';
print_r($array);

//（结果！！！）Array ( [0] => a [1] => b [2] => c [4] => e )

（注意！！！）删掉一个单元 键的值继续增长不受影响

$array['1'] = 'yi';
$array[1] = '一';
print_r($array);

//结果 Array ( [1] => 一 )

 

二、数组的循环

1.for循环

for ($i = 0, $i < count($array); $i++) {

for ($i = 0, $len = count($array); $i < $len; $i++) {  //这样效率高 否则每次都要调用count函数

}

2.foreach循环

像下面这样的 键没有规律的数组 就没法用for循环来遍历

foreach开始执行时 数组内部的指针会自动指向第一个单元

（1）键值都要

$array = array(3 => 'a', 6=> 'b', 'a' => 'c');

foreach ($array as $key => $value) {

}

echo $key,' => ', $value;  //（注意！！！）停留在了最后一次的值

var_dump(current($array));  //（注意！！！）bool(false);  -  它使劲往后循环 foreach结束之后 不会自动将数组reset（参见下面的游标操作）

（2）只要值

foreach ($array as $value) {

}

 

3.游标操作

数组内部有一个游标 指向当前正在操作的值

游标操作函数

current, key, next, prev, reset, end

 

（1）current  -  游标当前所指向的单元的值

$array = array('a', 'b', 'c', 'd');

echo current($array);  //a  -  数组刚创建 指针指向第0个单元

（2）key  -  游标当前所指向的单元的键

echo current($array);  //0  -  数组刚创建 指针指向第0个单元

（3）next  -  右边向后移动一个单元

echo next($array);  //b  -  移动并返回移动后的值

echo next($array);  //c
echo next($array);  //d
var_dump(next($array));  //bool(false)  -  next到数组的外面  使用next和current都返回false 使用key返回null
var_dump(current($array));  //bool(false) 
var_dump(key($array));  //null

（4）prev  -  指针向前移动一个单元

（5）end  -  将指针指向数组的最后一个单元

（6）reset  -  将指针指向数组的第一个单元

 

4.for循环 循环关联数组

for (; current($array); next($array)) {

}

（注意！！！）陷阱！！！

$array = array(0, 1, 2, 3);

for (; current($array); next($array)) {

echo current($array);

}

数组的第一个元素是0 在for循环进行current($array)的真假判断时 就是假 此方法根本就无法遍历这个数组

 

5.each

把指针指向的当前单元的信息组成一个数组 然后再把数组的游标向后移动一下

Array(

1 => 单元值,
'value' => 单元值,
0 => 键,
'key' => 键

)

如果游标移出了数组 会返回 bool(false)

while ($current = each($array)) {

}

 

6.list

80%的PHPer都没搞清楚

list可以把数组的元素 分别赋值给变量

$array = array('春', '夏', '秋', '冬');
list($apring, $summer, $autumn, $winter) = $array;
这样 春夏秋冬就分别赋值给了这四个变量

（注意！！！）在list中变量的顺序 分别对应数组的 [0], [1], [2], ...

list($a, $b, $c) = array(2 => '赵', 3 => '钱', 1 => '孙');

//结果：$a, $b, $c 分别是 null, 孙, 赵

（注意！！！）list的执行过程是 先产生映射 然后按照参数从右向左进行赋值

list($arr[0], $arr[1], , $arr[2]) = array(3 => '赵', 1 => '钱', 5 => '孙', 2 => '李');
print_r($arr);

//结果：Array ( [2] => 赵 [1] => 钱 [0] => )  -  是从右向左赋值的

 

7.each + list

each 经常和 list 配合使用

while ($temp = each($array)) {

list($key, $value) = $temp;

echo $key,' => ',$value;

}

 

三、数组的下标

整形：不必加引号 加了引号 php还要把它转成整形

字符串型：

不加引号 将被理解为常量 按照常量值做下标 如果找不到对应常量定义 直接当成下标 并且报Notice

单引号 效率高

双引号 会解析里面的变量 效率低

error_reporting(E_ALL);  //开启所有错误报告

$array = array(0 => '爱', 'yu' => '芋', '头' => '头');

echo $array[0],$array['0'],$array["0"];

echo $array[yu],$array['yu'],$array["yu"];

define('头', 'aaa');
$tou = '头';

echo $array[头],$array['头'],$array["$tou"];

 

/* 结果：

爱爱爱
Notice: Use of undefined constant yu - assumed 'yu' inD:\APMServ5.2.6\www\htdocs\study\phpbasic\array.php on line 48
芋芋芋

（1）count  -  数组元素个数 对象属性个数

int count ( mixed $var [, int $mode = COUNT_NORMAL ] )

null：返回0

其他：返回1

mode  -

3.数组判断函数

（1）array_key_exists  -  键是否存在

bool array_key_exists ( mixed $key , array $search )

docs/php/zh/function.array-key-exists.html

array_key_exists() 在给定的 key 存在于数组中时返回 TRUE。key 可以是任何能作为数组索引的值。 array_key_exists() 也可用于对象。

isset 和 array_key_exists 的区别

$array = array('a', null);

var_dump(array_key_exists(0, $array));  //bool(true)

var_dump(array_key_exists(1, $array));  //bool(true)

var_dump(isset($array[0]));  //bool(true)

var_dump(isset($array[1]));  //bool(false)  -  isset是根据值来判断的 如果有这个键 但是它的值是null 仍然返回false

isset不是函数 是语法结构 速度比 array_key_exists 快 但是isset是根据值来判断的

isset并不仅限于数组 可以判断变量是否设置

（2）in_array  -  某个值是否在数组中

bool in_array ( mixed $needle , array $haystack [, bool $strict = FALSE ] )

strict  -  为true 将用全等“===”进行检测

 

4.数组整体处理函数

（1）array_flip  -  交换键值

array array_flip ( array $trans )

如果存在相同的值 后交换的 会覆盖原来的交换值

$array = array('a' => 0, 'b' => 1, 'c' => 1, 'd' => 2);

print_r(array_flip($array));  //Array ( [0] => a [1] => c [2] => d )

非字符串和整形的不能转换 会卡住

$array = array('a' => 0, 'b' => array(1, 2));

print_r(array_flip($array));

SORT_REGULAR - compare items normally (don't change types)

$array = array('a', 'b', 'a');

（5）array_walk_recursive  -  递归操作数组

bool array_walk_recursive ( array &$input , callable $funcname [, mixed$userdata = NULL ] )

docs/php/zh/function.array-walk-recursive.html

将用户自定义函数 funcname 应用到 array 数组中的每个单元。本函数会递归到更深层的数组中去。

funcname  -

典型情况下 funcname 接受两个参数。input 参数的值作为第一个，键名作为第二个。

userdata  -

（3）sort  -  数组升序

bool sort ( array &$array [, int $sort_flags = SORT_REGULAR ] )

docs/php/zh/function.sort.html

本函数对数组进行排序。当本函数结束时数组单元将被从最低到最高重新安排。key 会重新排列 无论整形key还是字符串key

sort_flags  -

SORT_REGULAR - 正常比较单元（不改变类型）
SORT_NUMERIC - 单元被作为数字来比较
SORT_STRING - 单元被作为字符串来比较
SORT_LOCALE_STRING - 根据当前的区域（locale）设置来把单元当作字符串比较，可以用 setlocale() 来改变。
SORT_NATURAL - 和 natsort() 类似对每个单元以“自然的顺序”对字符串进行排序。 PHP 5.4.0 中新增的。
SORT_FLAG_CASE - 能够与 SORT_STRING 或 SORT_NATURAL 合并（OR 位运算），不区分大小写排序字符串。

$fruits = array('a' => "lemon", 'b' => "orange", 'c' => "banana", 'd' => "apple");
sort($fruits);
print_r($fruits);

//结果：Array ( [0] => apple [1] => banana [2] => lemon [3] => orange )

（5）ksort  -  按照键排序

bool ksort ( array &$array [, int $sort_flags = SORT_REGULAR ] )

6.队列/栈操作函数

队列：先进先出

人排队 先排的 先被选走

栈：后进先出

往缸里放东西 后放进去的能先拿出来

（1）array_push  -  将一/多个单元 放到数组尾部（压栈）

int array_push ( array &$array , mixed $var [, mixed $... ] )

返回压入成功后 数组单元的个数

$array = array();
array_push($array, '春');
array_push($array, '夏', '秋', '冬');
print_r($array);

 

//结果：Array ( [0] => 春 [1] => 夏 [2] => 秋 [3] => 冬 )

mixed array_pop ( array &$array )

docs/php/zh/function.array-pop.html

弹出并返回 array 数组的最后一个单元，并将数组 array 的长度减一。如果 array 为空（或者不是数组）将返回 NULL。 Will additionally produce a Warningwhen called on a non-array.

（3）array_unshift  -  将一个/多个单元 放在数组开头

int array_unshift ( array &$array , mixed $var [, mixed $... ] )

docs/php/zh/function.array-unshift.html

array_unshift() 将传入的单元插入到 array 数组的开头。注意单元是作为整体被插入的，因此传入单元将保持同样的顺序。所有的数值键名将修改为从零开始重新计数，所有的文字键名保持不变。

$queue = array("orange", "banana");
array_unshift($queue, "apple", "raspberry");
print_r($queue);

Array

    [0] => apple
    [1] => raspberry
    [2] => orange
    [3] => banana
)

mixed array_shift ( array &$array )

docs/php/zh/function.array-shift.html

array_shift() 将 array 的第一个单元移出并作为结果返回，将 array 的长度减一并将所有其它单元向前移动一位。所有的数字键名将改为从零开始计数，文字键名将不变。

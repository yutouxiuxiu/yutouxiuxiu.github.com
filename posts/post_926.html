在这里给大家介绍一个java 生成条形码

支持EAN13, EAN8, UPCA, UPCE, Code 3 of 9, Codabar, Code 11, Code 93, Code 128, MSI/Plessey, Interleaved 2 of 5, PostNet等

这个项目从2005年就成立到现在以有两个修正版本

一、Aptana插件

以前

没有提示

没有报错

现在

使用Aptana插件

1.下载

用aptana_update_024747

（因为新版的没办法离线安装）

还出了一个独立的IDE

2.安装

（1）制作插件包

解压缩到名为aptana

在里面新建 eclipse 文件夹

将 features 和 plugins 目录放入 eclipse 文件夹中

其他文件删除

此插件不支持绿色版的MyEclipse

此插件不支持安装了MyEclipse多个版本

（2）MyEclipse8.5以上

进入MyEclipse安装目录/MyEclipse 10/dropins/

将插件包复制进来

（3）MyEclipse8.5以下

将插件包复制到本地磁盘任意目录（不要有中文和空格）

进入MyEclipse安装目录/

新建 links 文件夹 并进入

新建 aptana.link 文件

path=aptana插件包的目录（””要写成”\”）

首次安装启动myeclipse报错问题

Aptana –> MyAptana/Message

选择 Never display after startup

取消 Use Firefox as the …

（4）Eclipse

此插件不支持Eclipse3.2和4.x版本

进入Eclipse目录

进入 dropins 目录

将插件包复制进来

也可以在线安装插件

3.验证是否安装成功

Window下多了两个 My Aptana 和 My Cloud

或者是Preference中有Aptana

4.配置插件

打开Preference

（1）使支持浏览器

Aptana –> Browsers/User Agent

勾选 IE、火狐

如果要使用火狐浏览器 必须在安装插件之前就存在火狐

（2）使支持jQuery

Aptana –> Editors –> JavaScript –> Code Assist

勾选jQuery

 

5.使用

ctrl + ? 注释

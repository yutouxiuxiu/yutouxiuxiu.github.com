EL表达式

{ post:time }
一、语法

${ EL表达式 }

二、概述

为了使JSP中尽量少出现<% %>  使页面布局和逻辑分开 方便界面的设计 就可以使用 EL 和 JSTL表达式

大公司应用的很多 中国的很多小公司就瞎搞了

Tomcat5.5之后才能用EL表达式 否则要在jsp中配置它不忽略EL表达式

三、作用

1.获取四个范围数据的值

用EL获取数据的好处

  -  如果要取的值不存在 EL会输出空的字符串”” 而传统方式是null

（1）取四个范围对象中变量的值

存在四个范围数据的值

pageContext.setAttribute(“pageInfo”,”pageValue”);

request.setAttribute(“requestInfo”,”requestValue”);

session.setAttribute(“sessionInfo”,”sessionValue”);

application.setAttribute(“applicationInfo”,”applicationValue”);

EL表达式取值：

${ pageScope.pageInfo }

${ requestScope.requestInfo }

${ sessionScope.sessionInfo }

${ applicationScope.applicationInfo }

EL的简写形式取值：

${ pageInfo }

${ requestInfo }

${ sessionInfo }

${ applicationInfo }

        这些是不同名的参数

       （注意！！！）如果四个范围的数据有同名的 简写的时候

          -  page < request < session < application 按照范围由小到大寻找 发现了就输出

             实质上是使用了pageContext中的findAttribute

                     这个方法就是按照这个顺序寻找值的

（2）取四个范围对象中数组中的值

pageContext.setAttribute(“country”, new String[] {“中国”, ”美国”, “日本”});

EL表达式取值：

${ country[1] }

这里边用“[ ]“取 是因为它是数字（下面总结里写的 什么情况下用什么取）

（3）取四个范围对象中集合中的值

List：

List<String> list = new ArrayList<String>();

list.add(“1”);

list.add(“2”);

pageContext.setAttribute(“country”, list);

EL表达式取值：

${ list[1] }

Map：

Map<String, String> map = new HashMap<String, String>();

list.put(“a”, ”芋头”);

list.put(“b”, ”修修”);

pageContext.setAttribute(“country”, map);

EL表达式取值：

${ map.a }

这里边用“.“取 是因为它是key（下面总结里写的 什么情况下用什么取）

（4）取四个范围对象中对象中的值

Person p = new Person();

p.setName(“芋头”);

p.setAge(“21”);

pageContext.setAttribute(“country”, p);

EL表达式取值：

${ map.name }

总结：

什时候用”[ ]”取值什么时候用”.”取值

① 如果是数字或者特殊字符（”.”，”-”）用 ”[ ]”

    例如 list.put(“bbb.ccc”, ”修修”);

        -   要用 ${ map[“bbb.ccc”] }取

② Map的key和类的属性 直接可以用”.”

（Double不能作为Map的key 因为它不精确）

2.执行运算

（1）数学运算

如果看你是数字的话 它会自动转成整形相加 而且再加一个不存在的也能自动忽略

但是这样就不行了 不能完成字符串的拼接

（2）关系运算

比较之后返回 true或false

（3）逻辑运算

判断之后返回 true或false

（4）判断是否为空

${ empty 变量 }

（5）三元运算符

${ empty user?”用户为空”:user }

 

3.操作web中的隐含对象

有11个隐含对象

（1）pageContext

这是一个入口 里边就封装了 九大内置对象

九大内置对象总结

内置对象 类型 作用域
request javax.servlet.http.HttpServletRequest request
response javax.servlet.http.HttpServletResponse response
pageContext javax.servlet.jsp.PageContext page
session javax.servlet.http.HtpSession session
application javax.servlet.jsp.ServletContext application
out javax.servlet.jsp.JspWriter page
config javax.servlet.ServletConfig page
page java.lang.Object page
exception java.lang.Throwable page

可以通过 get对象 来获取其他八大对象

（2）pageScope

（3）requestScope

（4）sessionScopr

（5）applicationScope

（6）param

相当于request.getParameter(“”);

${ param.username }

（7）paramValues

相当于request.getParameterValues(“”);

可以接收多个

${ paramValues.hobby[0] }

EL表达式能取一个 用JSTL可以实现循环

（8）header

获得头信息

相当于request.getHeader("");

${ header.host }

${ header["user-agent"] }

（9）headerValues

相当于request.getHeaders("");

返回的是一个数组

（10）cookie

${ cookie.username.value }

取出cookie里name的值

记住用户名密码的时候就可以用哦

（11）initParam

获得初始化参数 初始化参数需要在项目的web.xml中配置

<context-param>

    <param-name>username</param-name>

    <param-value>yutou</param-value>

</context-param>

相当于getServletContext().getInitParameter(“username”);

${ initParam.username }

 

4.调用Java函数

只有静态方法才能被EL调用

调用自定义方法

（1）写一个类 提供一个静态方法

（2）在WEB-INF下写一个.tld文件（标签库文件）

创建tld文件

更改自动生成的名称空间

在tld文件中 进行自定义方法的描述

（3）引入标签库 调用方法

引入自定义的标签库文件 调用自定义的方法

定义自带方法

在JSTL的jar包中META-INF中的fn.tld中定义了很多自带的方法 有转换大小写 字符串拆分 替换 截取 判断包含等字符串的功能

可以在jsp中引入此标签库

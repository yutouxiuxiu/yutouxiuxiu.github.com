PS - 软件操作

{ post:time }
一、基本操作

新建：ctrl + N

常用单位：只说要做一个200*200那就是毫米 做网页就用像素

分辨率：网页72 印刷品300

画面三种显示方式的切换：F

切换画布：ctrl + tab

隐藏工具箱、工具属性栏、悬浮面板：Tab

进入专家模式 让各种东西都没有

缩放：Z

CS5中 去掉细节缩放 可以实现放大到选中部分

切换到抓手：H

撤销：ctrl + alt + Z

暂时切换到抓手：空格

切换隐藏工具：shift + 相应工具的快捷键

裁剪：C

恢复到图像一开始打开的状态：F12

二、辅助

网格：ctrl + "

调节网格： 首选项（ctrl + K） -> 参考线、网格和切片（ctrl + 6/8） -> 更改网格间隔（要和画布的单位一致）

标尺：ctrl + R

从标尺上能拖拽出辅助线 用移动工具拖拽辅助线到外部可以删掉

隐藏辅助线：ctrl + :

三、选区

移动工具：V

选择：在移动工具下 按住ctrl 选中的部分的相应图层也会被选中

选区：控制操作的范围 

矩形选框工具：M

约束到正方形/正圆：按住shift 

约束中心点为鼠标落点：按住alt

取消选区：ctrl + D

选区布尔运算：

新选区：M / 拖动现有选区到新位置（移动过程中用shift约束水平/垂直移动）

相加：按住shift

相减：按住alt

相交：按住shift + alt

填充：

色彩的三要素：色相 饱和度 明度

前景色 背景色 拾色器

恢复到默认的前景色背景色：D

前景色背景色切换：X

填充前景色：alt + backspace

填充背景色：ctrl + backspace

变换选区：选择（alt + S） -> 变换选区（T） -> 调节 -> 回车确认

约束等比缩放：按住shift拖拽边角点

约束中心点不动：按住alt

获得物件的选区：（比如要用一个图形减去另一个图形的形状）

图层转选择：图层所有像素的边缘选区 按住ctrl在图层面板中想要获得图层的预览区单击

反选：ctrl + shift + I

四、图像处理常用方式

         剪版                      挖版                羽化版           出血版

      宁静、高雅风格         热情、嘈杂、廉价风格      女性化风格

羽化版：

画出选区 -> ctrl + alt + D（CS2）/ shift + F6（CS2/CS5） -> 输入羽化值

 

五、画笔

画笔：B

以当前的前景色进行绘制

笔尖大小：变小："[" 变大："]"

笔尖硬度：变软：shift + [ 变硬：shift + ]

如果在画布上点一个点 然后按住shift在下一个地方点击 会自动以直线连接

调节面板：F5

画笔形状：可以用各种图案进行绘制 可以从网上下载（.abr）比如说做旧的纹理划痕 然后在顶上的工具属性栏点击画笔 然后向右箭头里加载

间距：画笔画出来的线 是一堆紧密排列在一起的圆构成的 这个间距就是调节那些圆之间的距离的

形状动态：

大小抖动：构成直线的圆的大小 可以有抖动

角度抖动：构成直线的单元形状在画布平面上的角度进行抖动

圆度抖动：如果xy轴在画布上 圆度抖动就是向z轴方向抖动

散布：有的地方多 有的地方少 比如说下面这种效果

历史记录画笔

当画面整体进行了某些调节的时候 然后某些局部不想要调节之后的效果 想要之前的 就用历史记录画笔涂抹 那些地方就变成了之前的了

默认是恢复到画面的原始状态 可以通过点击特定历史记录前面的框 把历史记录画笔还原到的位置进行设定

历史记录保存数量

ctrl + k 首选项

可能在性能中 版本不一样 历史记录数量 专业的给到10就够了 默认是20 因为每次都要新建图层操作 否则恢复很多 后面的就都白做了

六、图层

新建图层：ctrl + shift + alt + N

删除图层：拉动图层到垃圾桶

到移动工具 -> 选中图层 -> delete

在有选区的情况下 保留选区并删除图层：alt + L -> L -> L （CS5中没有第二个L的快捷键）

合并图层：选择要合并的图层 -> ctrl + E

调节图层顺序：ctrl + [ 下移 ctrl + ] 上移

ctrl + shift + [ 到最底层 ctrl + shift + ] 到最顶层

投影效果：在图层面板的文本层上双击 

投影 不要打对号！ 点“投影”两个字 

不透明度 一般用75% 印刷出来比较不错

角度 30 120 -30 -120 看起来舒服 勾选使用全局光 这样会使整个画面中只有一个光源

效果的复制：按住alt 拖拽一个图层的效果到另一个图层中

复制选中图层效果 alt + L -> Y -> C

粘贴效果到选中图层 alt + L -> Y -> P

清除选中图层所有效果 alt + L -> Y -> A

背景层转化为图层：在背景层双击 按回车

快速隐藏很多图层：按住一个图层的眼睛 向下拖拽到每个想要隐藏的图层的眼睛上

整齐排列：选择几个要进行排列的图层 -> V移动工具 

图层锁定：

透明锁定：锁定之后 给图层填充的时候 只填充图层不透明的部分

七、复制

位移复制：选中要复制物件所在的图层 V移动工具状态下 按住alt 拖拽

原位复制：（比如说做靶）

法1 拖拽物件所在图层到新建图层上 就出现了两个完全一样的图层（特殊情况采用的 备用方案）

法2 选中物件所在图层 ctrl + j （很常用） 还可以用来备份 没改好直接删图层就行

如果有选区在 用ctrl + j 则 只复制选区中的内容到新的图层

复制选区到新的图层：ctrl + j

在有选区的情况下备份整个图层：拖拽图层到新建按钮

 

八、套索工具

套索工具：L 绘制曲线 按住拖拽 如果中途松手 它会自动以直线连接到起始点

多边形套索工具：绘制直线连接的多边形 

自动以直线闭合：在没有闭合的地方双击

取消上一次的定点：backspace

取消所有定点：esc

绘制过程中套索工具和多边形套索工具的切换：在套索或者多边形套索状态 按住alt 如果用单击方式操作 就是多边形套索 如果按住拖拽就是套索

磁性套索工具：自动吸附边缘 不用鼠标点击 跟随鼠标进行选区绘制 鸡肋 没啥用 做纯色背景挖版好使 但是魔术棒更简便 复杂背景挖版根本识别不了

 

九、文字工具

文字工具：T

文字颜色取决于前景色 如果一开始颜色不正确 就改前景色 针对文本层 重新填充 不要改上边工具栏中的颜色设置

文字都会自动新建图层（文本层）

完成文字输入：ctrl + 回车

取消文字输入：esc

重新编辑文字：直接选文字

双击文本层的预览区

调节字间距：全选文本状态 alt + ←/→

只想调节某两个字间的距离：把光标放在两个字中间 窗口 -> 字符 改文字间距

调节行距：全选文字状态 alt + ↑/↓

段落中不让行首出现标点：窗口 -> 段落

避头尾法则 = 宽松/严格

但是这样 文本的行尾就对不齐了 就点一下最后一行左对齐

段首空两格：段落里的首行缩进 改成文字大小点数的二倍

打文字往左扩展：在段落中右对齐

 

 

十、魔术棒

魔术棒：W 针对颜色边缘形状来创建选区

连续：选区是否连续 如果不要求连续 画面只要满足这个容差 都能被选进来 

 

十一、渐变工具

渐变工具：G

设置好前景色 和背景色 也就是从前景色过度到背景色

进行选区 可以使用辅助线进行限定

然后用shift约束进行拖拽

 

十二、颜色调节

比如说要改一个衣服的颜色

进行衣服的选区 -> 色相饱和度（ctrl + U） -> 拖拽色相 换成各种颜色都可以

 

十三、橡皮擦

橡皮擦：E

同样可以调节大小和强度还有不透明度

如果擦背景层 擦出来的颜色就是背景色

双击背景层 回车确定 转换成普通图层 再擦 就是透明的

十四、自由变换

自由变换：ctrl + T

只识别中心点：按住alt 用于图像比较窄 无法选到中心点

把中心点移到哪 旋转图像的时候 就根据哪个点进行旋转

在自由变换状态下 右键

斜切：按住ctrl + shift 调节边角点 只能按照边角点所引出的两个边的某个边进行调节

扭曲：按住ctrl 不受任何约束

透视：按住ctrl + shift + alt 拖拽一个点 对应的另一个点也跟着动 出现近大远小的样子

变形：

 

十五、仿制图章

仿制图章：S

连续纹理修复：按住alt点击纹理线 移动到要仿制的地方 瞄准纹理线 松开alt 进行仿制

修复带有透视关系的连续纹理：滤镜 -> 消失点 -> 绘制透视关系网格 蓝色的话就代表透视关系正确

 

十六、修复画笔工具

修复画笔工具：J

在完成遮盖的过程中 可以实现和背景之间颜色融合 修人脸上的黑点儿 豆豆 就很容易 不怕颜色差太多

污点修复画笔工具：不用取样 就一抹就行了 就直接融合背景

修补工具：

源：从外面拿到里头来

目标：从里面拿出去

透明：把边缘趋向于无用的像素直接透明处理

像下面这样图的修补 就直接用修补工具 极其方便 仿制图章会很难弄

红眼修复

在红的眼睛上单击一下就好了 原理就是把红的变成黑的 在别的红的上一点 你就发现了

十七、模糊工具

做画面的景深的感觉可以用

涂抹工具：

可以用来画烛光

十八、减淡

减淡就会让它变亮 一直亮下去就白了

加深就是让它变暗 慢慢就黑了

海绵工具 让饱和度降低 抹抹慢慢就变黑白了

十九、渐变工具

渐变工具：G

线性渐变 径向渐变 角度渐变 对称渐变（做UI用） 菱形渐变

可以用来画彩虹 然后用自由变换工具变形调整弯度

可以用来画金箍棒

可以画金属球

一、卸载MySQL

1.在控制面板里先卸载

2.MySQL5.5版本之后 数据文件和安装目录不在一起 也删掉 在Document and Files/All Users/Application Data里

二、安装MySQL

默认端口号：3306

默认字符集：latin1（就是iso8859-1）

选择字符集 utf8

要勾选上Windows PATH 好让在命令提示符中识别MySQL的命令

不要勾选让远程登录root用户

这样就说明安装成功

登录mysql命令

mysql (默认：–h localhost –P 3306) –u root –p

-h 主机

-P 端口

-u 用户

-p 密码（回车之后可提示输入密码）

 

三、MySQL重置root密码

1.修改方式

通过非服务方式加skip-grant-tables运行mysql来修改mysql密码
（1）停止mysql服务
（2）跳过管理员启动mysql

    打开命令行窗口，在bin目录下使用mysqld-nt.exe（5.5为mysqld.exe）启动，即在命令行窗口执行:

mysqld-nt --skip-grant-tables（5.0的命令）

mysqld –skip-grant-tables（5.5的命令）

（3）登录mysql

    另外打开一个命令行窗口（不要关闭刚才的！）

mysql –u root –p

    登录mysql，此时无需输入mysql密码即可进入。

    然后使用mysql数据库

use mysql

（4）改密码

update user set password=password(‘新密码’) where user=’root’;

（5）启动MySQL服务

2.如何停止/启动MySQL服务
运行 –> services.msc

（或者我的电脑右键 –> 管理 –>服务和应用程序 –>服务）

停止/启动MySQL服务

 

四、将以前的MySQL数据文件迁移到新的MySQL数据库中

先要在新的MySQL数据库中创建和原来数据库名称相同的数据库

mysql> create database 数据库名

然后将原来数据库data目录下这个数据库的文件（注意不是文件夹整个拷贝 新的MySQL创建数据库时 可能会产生一个db.opt文件）拷贝到新的数据的对应目录下

 

五、MySQL的存储结构

1.MySQL中可以创建多个数据库

    一般情况下 每个软件创建一个单独的数据库

2.每个数据库中可以创建多个数据表

3.数据表就相当于二维表格 相当于java的类

3.数据表中的每条记录 相当于对应的java对象

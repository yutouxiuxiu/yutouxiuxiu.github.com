一、php的字符串

在开发网站的过程中 想对字符串做什么操作 基本上php都提供了

字符串的函数非常非常的丰富

字符串在开发和面试的时候也是最频繁用到的

 

二、声明字符串

1.双引号

解释转义字符 \" \' \t \n(换行) \r(回车) \$(不让解释成变量值) \\

2.单引号

除了 \\ 进行转义 其他一律原样输出

不对变量做解析

↑.两者区别

（1）执行速度不同

单引号快

（2）转义内容不同

↑.如何选择

大段文本 使用单引号 解析快

确实要再一段字符串中夹杂变量 用双引号 比如拼接SQL语句时

 

3.定界符

如果一大段文本 里边有很多行 里面既有单引号 还有双引号 导致转义麻烦

可以使用 heredoc, nowdoc（PHP5.3.0+） 两种方式

（1）heredoc

像双引号一样解析大段文本

<<<标识符
文本内容
标识符;  //这个必须要独占一行 前面没有其他任何字符 空格也不行

例如

$a = '变量值';
$str = <<<title
文本内容
' " \ \\
\t \r
$a
title;

echo $str;
/* 结果：文本内容 ' " \ \ 变量值 */

注意：

标识符和变量命名规则一致

对文本的处理和双引号一样 转义很多字符 并且解析变量

结束符要独占一行 且前面没有任何字符

（2）nowdoc 像单引号一样解析大段文本

PHP5.3.0之后新增的

写法就是在上边的标识符上加上单引号

<<<‘标识符’
文本内容
标识符;

1.字符串长度函数

（1）int strlen($str)

按照字节算的

中文

utf-8  -  1个汉字3个字节（.php文件是utf-8编码）

gbk  -  1个汉字4个字节（ANSI代表gbk）

（2）mb_strlen(string $str[, string $encoding])

按照编码返回字符串长度 “码表_strlen”

echo mb_strlen("芋头修修", "utf-8");  //4  -  它内部应该就是把实际字节长度/3 所以如果指定为gbk 它就得到6

2.查找字符串中子字符串函数

（1）查找位置

strpos(string $haystack, mixed $needle[, int offset=0])

haystack  -  在该字符串中查找

needle  -  要找的

mixed  -  混合类型 可以容纳多种类型参数 如果是整形 会被理解为ascii码

offset  -  偏移量

return int/false

var_dump(strpos('12345678a', 'a'));  //int(8)
var_dump(strpos('12345678a', 345));  //bool(false)
var_dump(strpos('12345678a', 97));  //int(8)
var_dump(strpos('12345678a1234a', 97, 8));  //int(4)  -  在偏移的基础上的位置 并不是从字符串头开始的 4+8=12

（注意！！！）如果要查找的在第一位 会返回0 如果用if来判断 是false的结果 请用“===”判断

（2）stripos

和（1）一样 但这个对大小写不敏感

（3）strrpos

和（1）一样 但这个是查找最后一次出现的位置

（4）查找出现次数

echo substr_count(string $haystack, mixed $needle[, int offset=0[, int $length]])

 

3.字符串替换函数

（1）str_replace 

mixed str_replace ( mixed $search , mixed $replace , mixed $subject [, int &$count ] )

docs/php/zh/function.str-replace.html

count  -  可以传一个变量进去 然后会把替换的次数记录到这个变量中

可以一次性替换多个

$search = array('hi', 'heihei');

$replace = array('hello', 'hehe');

str_replace($search, $replace, 'hi, this is heihei');

如果要把hi替换成hello 但是this单词里也有hi怎么办？

用正则表达式

想把hi变成hello hello变成hi 怎么办？

中间变量 hi -> [#asdljwoncolcmosdfjw]

hello - > hi

[#asdljwoncolcmosdfjw] -> hello

（2）str_ireplace

忽略大小写的版本

docs/php/zh/function.str-ireplace.html

（3）strtr

string strtr ( string $str , string $from , string $to )

string strtr ( string $str , array $replace_pairs )  //这种方式看起来清晰

docs/php/zh/function.strtr.html

把出现在 from 中的字符 一个一个转换为 to 中相应的字符。

如果 from 与 to 长度不相等，那么多余的字符部分将被忽略。

$a = 'abcdef';
echo strtr($a, 'ab', 'bc');  //bccdef    -  a->b, b -> c

全角转半角

$str = '１２３４５';

echo strtr($str, array('１' => '1', '２' => '2', '３' => '3'));

ecshop中就有这样的函数

（4）substr_replace  -  将字符串指定部分 替换

mixed substr_replace ( mixed $string , mixed $replacement , mixed $start [, mixed $length ] )

docs/php/zh/function.substr-replace.html

substr_replace() 在字符串 string 的副本中将由 start 和可选的 length 参数限定的子字符串使用 replacement 进行替换。

 

4.提取子字符函数

（1）substr  -  截取字符串指定部分

string substr ( string $string , int $start [, int $length ] )

docs/php/zh/function.substr.html

返回字符串 string 由 start 和 length 参数指定的子字符串。

start  -  正数：前面跳过几个, 负数：要后几个

length  -  正数：截取的长度, 负数：截取的结束位置 从后往前数跳过几个

面试 中英文混合截取乱码问题

 

（2）strstr（=strchr）  -  截取字符串中 字符串第一次出现的位置到结尾

string strstr ( string $haystack , mixed $needle [, bool $before_needle = false ] )

docs/php/zh/function.strstr.html 

返回 haystack 字符串从 needle 第一次出现的位置开始到 haystack 结尾的字符串。

before_needle  -  true：返回之前的 false：返回之后的 （PHP > 5.3.0）

找文件后缀（.hbm.xml）

文件名

（3）stristr

strstr的忽略大小写版本

（4）strrchr  -  截取字符串中 字符串最后出现的位置到结尾

string strrchr ( string $haystack , mixed $needle )

docs/php/zh/function.strrchr.html

该函数返回 haystack 字符串中的一部分，这部分以 needle 的最后出现位置开始，直到 haystack 末尾。

找文件扩展名（.xml）

还可以用 pathinfo 系统函数  -  返回文件路径的信息

mixed pathinfo ( string $path [, int $options = PATHINFO_DIRNAME | PATHINFO_BASENAME | PATHINFO_EXTENSION | PATHINFO_FILENAME ] )

$pathinfo = pathinfo('yutou.love.xml');

5.分割、连接、反转函数

（1）str_split  -  将字符串 定长切成数组

array str_split ( string $string [, int $split_length = 1 ] )

将一个字符串拆成数组。

（2）split  -  按照正则 分割字符串

array split ( string $pattern , string $string [, int $limit ] )

docs/php/zh/function.split.html

preg_split() 函数使用了 Perl 兼容正则表达式语法，通常是比 split() 更快的替代方案。如果不需要正则表达式的威力，则使用 explode() 更快，这样就不会招致正则表达式引擎的浪费。

本函数返回一个字符串数组，每个单元为 string 经区分大小写的正则表达式 pattern 作为边界分割出的子串。如果设定了 limit，则返回的数组最多包含 limit 个单元，而其中最后一个单元包含了 string 中剩余的所有部分。如果出错，则 split() 返回 FALSE。

（3）explode  -  使用一个字符串 分割另一个字符串

array explode ( string $delimiter , string $string [, int $limit ] )

docs/php/zh/function.explode.html

此函数返回由字符串组成的数组，每个元素都是 string 的一个子串，它们被字符串 delimiter 作为边界点分割出来。

（4）implode  -  将数组 用特定的分隔符 连接成字符串

string implode ( string $glue , array $pieces )

string implode ( array $pieces )

docs/php/zh/function.implode.html

Join array elements with a glue string. 将数组用特定的分隔符连接成字符串

 

6.去除、增加空格函数

（1）trim  -  去除两边 多余字符

string trim ( string $str [, string $charlist ] )

docs/php/zh/function.trim.html

此函数返回字符串 str 去除首尾空白字符后的结果。如果不指定第二个参数， trim() 将去除这些字符：

" " (ASCII 32 (0x20))，普通空格符。
"\t" (ASCII 9 (0x09))，制表符。
"\n" (ASCII 10 (0x0A))，换行符。
"\r" (ASCII 13 (0x0D))，回车符。
"\0" (ASCII 0 (0x00))，空字节符。
"\x0B" (ASCII 11 (0x0B))，垂直制表符。

（2）ltrim

只删除左边的版本

（3）rtrim

只删除右边的版本

（4）chunk_split  -  每隔几个字符 加一个字符串

string chunk_split ( string $body [, int $chunklen = 76 [, string $end = "\r\n" ]] )

docs/php/zh/function.chunk-split.html

使用此函数将字符串分割成小块 非常有用。例如将 base64_encode() 的输出转换成符合 RFC 2045 语义的字符串。它会在每 chunklen 个字符后边插入 end。

每隔几个字符 加一个字符串

（4）str_pad  -  字符串两端补白 到指定长度

string str_pad ( string $input , int $pad_length [, string $pad_string = " " [, int $pad_type = STR_PAD_RIGHT ]] )

docs/php/zh/function.str-pad.html

该函数返回 input 被从左端、右端或者同时两端被填充到制定长度后的结果。如果可选的 pad_string 参数没有被指定，input 将被空格字符填充，否则它将被 pad_string 填充到指定长度。

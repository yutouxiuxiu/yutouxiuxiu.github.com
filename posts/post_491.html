cookie不支持中文的问题解决

{ post:time }

在默认的情况下cookie存入中文 会出现乱码

存入方法：

URLEncoder.encode(“中文”,”UTF-8”);

然后再存入

取出方法1：

取出的时候进行解码便可

URLDecoder.decode(cookie,”UTF-8”);

取出方法2：

用EL表达式调用函数

取出方法3：

用js完成解码（js里支持EL表达式 和<% %>）

在body 中 onload=”init()”

function init() {

    var name = “${ cookie.username.value }”;

    name = decodeURIComponent(name);

    写入到html中

}

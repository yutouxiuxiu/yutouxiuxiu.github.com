{ post:time }
一、静态

创建了一个类的多个对象时

创建几个对象 它的属性值就有多少个

方法只有一个

1.静态属性

有的属性是对象共同的 不需要那么多 所以用静态

静态的属性 是在类中的 类声明了 就存在 不依赖于对象访问

2.静态方法

普通方法和静态方法在内存中都只有一份

但是

普通方法需要绑定this 需要用对象来调用

静态方法不需要绑定this 通过类名可以调用

非静态的方法居然也可以用类名调用 不要这样做

error_reporting(E_ALL | E_STRICT);
class Yutou {
    public function kiss() {
        echo '啵啵';
    }
}

Yutou::kiss();

/*
Strict Standards: Non-static method Yutou::kiss() should not be called statically in D:\APMServ5.2.6\www\htdocs\study\phpbasic\static.php on line 9
啵啵
*/

但是如果方法体中有this 就出现致命错误 不知道this指定的是哪个对象

在PHP5.3的strict级别 PHP5.4的默认级别中 都会进行提示

二、语法

1.定义静态成员

在属性/方法最前加 static 修饰

static public ...

 

2.访问静态属性

类名::属性名

 

三、静态、非静态间的访问

静态不可以访问非静态

非静态可以访问静态

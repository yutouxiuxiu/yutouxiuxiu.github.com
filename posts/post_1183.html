1.如何引用

language  -  旧的 以前这么用

2.在哪引用

其实放哪都行 但是有区别

（1）head中

一般引入外部独立js文件放在这

（2）body前 head后

获取页面中的元素：不行

    但是有办法解决

定义函数但不调用：可以

定义函数而且调用：可以（要与页面元素无关）

三、BOM

1.navigator对象

比如说一些浏览器的基本信息 使用什么浏览器等 都是BOM里存着的

2.window对象

是最顶层的对象 代表窗口

（1）document

都是数组对象 因为可能有很多

链接对象

锚对象

表单对象

图片对象

（2）frame

页面的框架

（3）location

地址栏中的地址

（4）history

历史记录

forward back 前进后退

（5）screen

获取窗口的宽高

 

特点

每个对象都有属性、方法、事件

 

四、DOM

1.定义

可以是页面动态变化

把页面分为数据和结构

所有的html标签都是结构

用树形来表现

XML HTML XHTML CSS

2.DOM的级别

一级  -  核心和HTML

二级  -  XML

三级  -  载入和保存的方法

零级  -  并不是W3C标准

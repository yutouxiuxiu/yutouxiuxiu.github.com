四、事件

解决了浏览器的兼容问题

1.绑定事件

.事件名(function(){});

.bind(事件类型, function(){});  //针对已经存在的元素

2.取消事件

.unbind(事件类型);

3.实时追加事件绑定

.live(事件类型, function(){});

和bind的区别 事件委派 给匹配的元素添加事件 即使这个元素是以后添加上来的也有效

新元素只要符合规则 就具有这个事件

.die(事件类型);  //解除委派

4.事件触发

.trigger(事件类型);

$(function) {

$("#username").blur(function() {

alert(1);

});

$("#button").click(function() {

//在一个事件中去触发另一个事件

$("#username").trigger("blur");

})

}

。。。

<input type="text" id="username" />

<input type="button" id="button" value="按钮"/>

5.阻止默认事件

$(function(){

$("#del").click(function(event) {

var isConfirm = window.confirm("删除否？");

if (!isConfirm) {

event.preventDefault();  //阻止默认的href事件

}

});

})

。。。

<a id="del" href="del">删除</a>

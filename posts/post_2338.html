如何用smack api发送广播

{ post:time }

使用openfire的广播功能首先需要在插件的页面里安装broadcast,然后通过Smack就可以发送广播了.  

  1. Connection conn = new XMPPConnection("XMPP服务器");  
  2. conn.connect();  
  3. conn.login("admin", "密码");  
  4. Message msg = new Message("all@broadcast.XMPP服务器");  
  5. msg.setBody("this is a broadcast.");  
  6. connection.sendPacket(msg);

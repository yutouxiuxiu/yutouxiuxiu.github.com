php面向对象 - 概述

{ post:time }
一、定义类时需要注意的

1.如果属性在定义类的时候不初始化 就是NULL

2.属性初始化的时候必须是直接的值（不能使表达式）  -  和java不同 java里可以是运算结果

3.类中的方法 是在类范围内的函数 可以和外部的重名

3.如果外部和内部方法同名 直接调用调用的是外面的 加“$this->”是调用自身的

 

二、构造函数&析构函数

1.构造函数

public function __construct() {

//会在new的时候执行

}

（1）php中不能有多个构造函数  -  和java不同

（2）new的时候传的参数 也和 __construct中定义的参数一致

2.析构函数

public function __destruct() {

//会在对象被销毁的时候执行

}

（1）对象何时销毁

显式销毁

系统回收  -  当代码执行到最后一行的时候 所有申请的内存都要被释放掉 如果不显式销毁 php解释完了之后 也就自动销毁掉了（自动销毁也是执行析构函数）

$a = new Person();

unset($a);  //对象销毁

$a = 'hhh';  //对象销毁

$a = null;  //对象销毁

（2）关于引用传值

$a = new Person();

$b = $c = $a;

unset($a);

echo '<hr />';

何时执行Person的析构函数

$a.$b,$c都会共同指向一个object 即使unset掉了a object还是在的

所以只会在最后 执行一次析构函数

$a = $b = $c = new Person();

unset($a);

unset($b);

unset($c);  //在此时执行析构函数

 

三、this的绑定

当一个对象 调用方法时

该方法执行之前 会把$this绑定到调用此方法的对象

 

方法体内想要访问调用者的属性 必须用$this  -  和java不同 

如果不加 则被理解为方法内部的局部变量

一、array

1.every 测试数组

every( callback, [thisObject] ) : Boolean

用callback函数遍历一个数组 测试它的每个元素 测试通过返回true 不通过返回false

callback接收遍历到的数组元素 callback函数第一次返回false的时候 跳出

例如 检测数组中的元素是否都是正数

var arr = [1, 2, 3, -1 ,-2];

arr.every(function(item) {

if (item > 0) {

return true;

} else {

alert(item);

return false;

}

});

2.filter 过滤数组

filter( callback, [thisObject] ) : Array

用callback函数循环过滤数组 callback函数接收一个参数作为待过滤数组的元素 当callback返回true 数组元素留下 返回false 去掉该元素

3.join 用字符串连接数组元素成字符串

join( separator ) : String

Joins all elements of an array into a string.

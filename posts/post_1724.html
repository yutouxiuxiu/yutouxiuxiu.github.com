Zend framework - 概述

{ post:time }
一、什么是Zend framework

是一个MVC框架

​

目前流行的php框架有

Zend framework（简称ZF Zend公司的 庞大 功能强悍 难学）

thinkphp（中国开发的 国内开发的有一个毛病 如果它不赚钱 它就不升级了）

yii（外企用的比较多）

cakephp（轻巧）

还有很多公司 是用自己开发的框架 不用这些产品

1.好处

程序更加规范

提高开发效率

程序可读性增强（目录结构都规定了）

程序可维护性增强

2.不足

配置复杂

 

二、快速入门

1.使用工具创建项目

强烈建议在 PHP5.2 以上

也可以手动创建 但是太麻烦 没什么技术含量

使用zf.bat进行创建

解压目录/bin/zf.bat

cmd：

zf.bat create project 项目全路径

这里注意 要把PHP的根目录配置到系统环境变量PATH中才有效

2.目录结构

3.使用zend studio创建空php项目

和刚才创建的名称保持一致

不用zend studio直接创建zend项目 因为可能会出现问题 公司里也不推荐这种方式创建

4.把刚才创建的项目拷贝过来

5.引入zend framework库到项目中

压缩包/library/

把里面的Zend文件夹 复制到项目中的library目录

6.配置apache和php 使其支持zf框架

（1）确定php>5.2.0

（2）开启PDO和PDO相关数据库引擎

php.ini

extension=php_pdo_mysql.dll  去掉前面的分号

（3）开启Apache的rewrite模块

httpd.conf

①

LoadModule rewrite_module modules/mod_rewrite.so  去掉前面的井号

②开启识别.htaccess功能

AllowOverride None 改为 All

（注意！！！）可能有三处 都改成All 虚拟主机配置里的不管

7.配置虚拟主机

配置虚拟主机到项目的public目录下

httpd.conf

# Virtual hosts
Include conf/extra/httpd-vhosts.conf  这个的注释打开

Apache\conf\extra\httpd-vhosts.conf

<VirtualHost *:80>

DocumentRoot "D:/APMServ5.2.6/www/htdocs/yutouzend/public"  建议粘贴过来的话 斜杠换成前面的方向

ServerName yutouzend.com

DirectoryIndex index.php

<Directory />

options FollowSymLinks

AllowOverride None

Order allow,deny

Allow from all

</Directory>

</VirtualHost>

如果使用的是集成开发环境 例如APMServ 要把虚拟主机的读写权限改到整个项目的路径 而不是public 否则会报错 说无法读取。。。

hosts

添加DNS

127.0.0.1    yutouzend.com

8.访问

http://yutouzend.com

如果Apache是8080端口 就在地址后直接加:8080就行了 前面的配置都不变

看到下面的画面 就说明成功了！

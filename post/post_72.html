XML的约束 DTD约束

{ post:time }
  • 引入DTD

1.独立文件

DTD文档必须存成UTF-8 DTD文件的第一行要写 <?xml version="1.0" encoding="UTF-8"?>

本地 <!DOCTYPE 根标签 SYSTEM ".dtd"> 网络 <!DOCTYPE 根标签 PUBLIC "DTD名称" "http://DTD的url">

2.文件内部

一般不用

  • 书写步骤

先统计有多少个标签

然后再看标签中要出现的具体的内容

  • 语法
1.元素（标签）
<!ELEMENT 元素名称 使用规则>

使用规则（标签中可以出现的内容）：

(#PCDATA)  文本数据 EMPTY  必须空 ANY  任意 (子元素关系式)  可以出现的子元素

子元素  -  单个子元素

子元素,子元素  -  有顺序的

子元素|子元素  -  或（只能出现其中一个）

子元素后还可加出现次数限定符

+  -  一次或多次（比它多）

?  -  零次或一次（又还是没有啊）

*  -  零次或一次或多次（随便）

2.属性
<!ATTLIST 所属标签名称

属性名 属性类型 属性约束

...

>

属性类型

CDATA  -  文本数据 ID  -  不能重复（不一定是数字） (值1|值2...)  -  枚举

属性约束

#REQUIRED  -  必须的 #IMPLIED  -  可选的 #FIXED "固定值"  -  固定值 写XML时 必须把值写成它 "默认值"  -  默认值  如果没写那个属性 就带默认的
3.实体

就是给实际的内容创建一个别名 然后在XML里可以引用它

<!ENTITY 实体名称 "实体内容">

引用方式

&实体名称;

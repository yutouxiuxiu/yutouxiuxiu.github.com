Set<Map.Entry<String,String>> entrySet = map.entrySet(); Iterator <Map.Entry<String,String>> it = entrySet.iterator(); while (it.hasNext()) { Map.Entry<String,String> me = it.next(); String key = me.getKey(); String value = me.getValue(); }

Map.Entry 其实Entry也是一个接口 是Map的内部接口 interface Map { public static interface Entry //只有接口在成员位置的时候才能加静态修饰符 { public abstract Object getKey(); public abstract Object getValue(); } class HashMap implements Map { class HaHa impelements Map.Entry { public Object getKey(); public Object getValue(); } }
  • HashTable 哈希表（不怎么用了）
底层是哈希表数据结构（存进去没什么顺序）是这个数据结构 就得覆写 hashCode 和 equals 方法 不可以用空作为键/值 用作键的 该集合是线程同步的 jdk1.0
  • HashMap
底层是哈希表数据结构 允许null作为键/值 该集合线程不同步 jdk1.2 较HashTable效率高
  • TreeMap
底层是二叉树数据结构 线程不同步 可以用于给键进行排序（和Set很像 其实Set集合底层就是使用了Map集合）
